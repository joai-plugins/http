# HTTP

Connect HTTP to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/http) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-http@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-http --url https://cortex.joai.ai/mcp/apps/http
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/http.git ~/.cursor/plugins/local/joai-http
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-http&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktaHR0cCI6eyJ1cmwiOiJodHRwczovL2NvcnRleC5qb2FpLmFpL21jcC9hcHBzL2h0dHAifX19) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/http
```
